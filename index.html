Welcome to Project X

Our IP project, Project X , is an innovative robotic system developed by a team of seven talented Computer Science students. It combines advanced technologies like Wi-Fi (ESP8266), Bluetooth, facial expressions, and real-time video streaming, making it an exciting and interactive experience.

Robot Features

Project X is packed with exciting features that make it more than just a robot. It’s designed to be intuitive, engaging, and full of possibilities. Check out its incredible functions:

Movable with Phone

Control the robot's movement directly from your smartphone using the ESP8266 Wi-Fi module .

Camera Integration

View live streaming and playback with the robot's camera for easy remote monitoring.

Bluetooth Connectivity

Connect the robot via Bluetooth for short-range operations and interactive speech.

128x64 Display

Monitor the robot's status and outputs on a clear, small 128x64 display .

Products Used in Project X

Female to Female DuPont Line (40 Pin, 30cm)

Essential for connecting various components within the robot.

NodeMcu ESP8266 V3 Lua CH340 Wifi Dev. Board

Used for Wi-Fi connectivity to control the robot via the mobile app.

L298N 2A Based Motor Driver Module

Controls the motors and provides the necessary power for movement.

0.96 Inch I2C OLED Display

Displays real-time system information like battery status and robot health.

Arduino Uno R3 CH340G

Serves as the main development board for controlling the robot's basic functions.

ISD1820 Recording Module Voice Board

Records and plays back audio for the robot to speak and respond.

82D12 12V Digital Power Channel Amplifier Bluetooth Speaker Module

Enables Bluetooth communication and audio playback for speech synthesis.

PAM8403 HW104 5V Audio Amplifier

Amplifies audio signals for enhanced speech output.

3 Inch 4Ω 10W Power Audio Woofer Speaker

Delivers high-quality sound for speech and audio playback.

DC 200RPM 3-6V BO Gear Motor

Drives the movement of the robot with precision and speed.

Li-Ion 18650 3.7V 2200MAH Battery

Provides power to the robot for long-lasting performance.
